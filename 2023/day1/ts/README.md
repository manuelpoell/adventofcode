# Advent of Code 2023 Day 1 - TypeScript solution

## Requirements

NodeJS v20.x.x

## Install dependencies

```shell
npm install
npm run build
```

## Run script

### Part one

```shell
node dist/index.js -f /path/to/input_file
```

### Part two

```shell
node dist/index.js -e -f /path/to/input_file
```
