# Advent of Code 2023 - Golang solutions

## Requirements

Golang v1.21.x

## Run script

Substitute {day} with the number of day you want to solve.

Substitute {file} with the path to the input file to use.

### Part one

```shell
go run . -d {day} -f {file}
```

### Part two

```shell
go run . -d {day} -f {file} -p
```
